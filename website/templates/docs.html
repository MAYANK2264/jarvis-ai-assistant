Getting Started

Jarvis AI is a powerful personal assistant that helps you control your computer using voice commands or text input. This guide will help you get started with Jarvis AI.

System Requirements

  • Windows 10/11, macOS, or Linux
  • Python 3.8 or higher
  • 4GB RAM minimum
  • Microphone (for voice commands)

Installation

Choose your preferred installation method:

Using pip (Recommended)

pip install jarvis-ai

From source

git clone https://github.com/MAYANK2264/JarvisAI.git
cd JarvisAI
pip install -r requirements.txt

Basic Usage

Start Jarvis AI by running:

python -m jarvis

Quick Start Commands

  • jarvis hello - Test if Jarvis is responding
  • jarvis help - Show available commands
  • jarvis exit - Close Jarvis AI

Customization

Create custom commands by extending the base Command class:

from jarvis import Command

@Command.register
def my_custom_command(text):
    """
    Custom command implementation
    Args:
        text (str): The command text
    Returns:
        str: Response message
    """
    # Your custom logic here
    return "Command executed successfully"

API Reference

Integrate Jarvis AI into your Python applications:

from jarvis import JarvisAI

# Initialize Jarvis
jarvis = JarvisAI()

# Execute commands
response = jarvis.execute_command("open chrome")

# Get available commands
commands = jarvis.get_commands()

# Add custom command handler
@jarvis.command_handler("custom_command")
def handle_custom(text):
    return "Custom command executed"

Troubleshooting

Voice Recognition Issues

  • Check microphone connection
  • Verify microphone permissions
  • Test in a quiet environment
  • Update audio drivers

Performance Issues

  • Check system requirements
  • Close unnecessary applications
  • Update to latest version
  • Try offline mode

Common Errors

  • ModuleNotFoundError: Install missing dependencies
  • PermissionError: Run with admin rights
  • TimeoutError: Check internet connection
