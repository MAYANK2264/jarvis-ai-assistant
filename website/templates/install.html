Installation Guide

Get started with Jarvis AI in minutes.

Prerequisites

  • ✅ Python 3.8 or higher
  • ✅ pip (Python package installer)
  • ✅ Git (optional, for development)

Quick Install

Install Jarvis AI using pip:

pip install jarvis-ai

Manual Installation

Clone the repository and install dependencies:

git clone https://github.com/MAYANK2264/JarvisAI.git
cd JarvisAI
pip install -r requirements.txt

Configuration

Create a configuration file at ~/.jarvis/config.yml:

language: en
voice_enabled: true
offline_mode: false
custom_commands_dir: ~/.jarvis/commands

Verify Installation

Test your installation by running:

python -m jarvis --version
